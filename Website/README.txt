Project: CRC

Description

This project showcases the impactful initiatives and completed projects of our organization. It includes features like dynamic project loading from a database, a secure donation form, and other essential pages.

Getting Started
Prerequisites

Ensure you have the following installed on your system:

- Python 3.8 or later
- MySQL or a compatible database server
- Setup Instructions

1. Navigate to the project folder

cd <path-to/project-folder>

2. Set Up Virtual Environment

python -m venv venv
source venv/bin/activate       # For macOS/Linux
venv\Scripts\activate          # For Windows

3. Install Dependencies
Install the required Python packages:

pip install -r requirements.txt


4. Run the Application
Start the Flask development server:

flask run

or

Manually run the 'project_folder/app.py' python file in File Explore 

5. Navigate to the Home Page
Access the home page in your browser by navigating to:

http://127.0.0.1:5000/home

Email: admin_@crc.org

6. Troubleshooting
If dependencies fail to install, ensure you are using the correct Python version.
Verify your database credentials and connection.
For further assistance, feel free to contact the developer.
